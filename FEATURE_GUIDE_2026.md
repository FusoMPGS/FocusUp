## 💻 DIGITAL WELLNESS SECTION

### NOTIFICATION MANAGEMENT
**The Problem**: Notifications interrupt focus
**The Solution**:
- Turn on "Do Not Disturb" (20-25 min sessions)
- Each notification costs 15-25 minutes of focus recovery
- Silence all notifications during study

**Recommended Tools**:
- iPhone: Settings → Focus → Study
- Android: Settings → Do Not Disturb
- Computer: System settings → Focus mode

### WEBSITE BLOCKERS
**Why Use Them?**: Social media is engineered for addiction
**Recommended Tools**:
- **Freedom**: Blocks distracting sites across devices
- **Cold Turkey**: Nuclear option for distraction
- **Forest**: Gamified focus with real tree planted
- **LeechBlock**: Simple, free website blocker
- **StayFocusd**: Chrome extension blocker

**Setup**: Block these during study
- Social media (Instagram, TikTok, Twitter, Snapchat)
- YouTube (unless needed for study)
- Gaming sites
- Entertainment streaming

### 20-20-20 RULE
**What It Is**:
- Every 20 minutes of screen time
- Look 20 feet away
- For 20 seconds
- Repeat!

**Why It Works**:
- Reduces eye strain
- Prevents headaches
- Improves focus return
- Natural eye relaxation

**How to Apply**:
- Set timer reminder every 20 min
- Look out window at landscape
- Or close eyes and rest
- Return refreshed to study

### SOCIAL MEDIA STRATEGY
**Don't Ban Completely**: Just manage it
1. **Designated Time**: Check social media at specific times
2. **Time Limit**: Set 15-minute windows only
3. **After Study**: Reward with social media browsing
4. **Separate Device**: If possible, phone in another room
5. **Accountability**: Tell friend you're avoiding it

## 🧠 ADVANCED STUDY TECHNIQUES

### 1️⃣ ACTIVE RECALL
**What It Is**: Testing yourself instead of just reading

**How It Works**:
- Read textbook chapter
- Close book
- Write down everything you remember
- Check what you missed
- Repeat until you remember 90%

**Why It Works**:
- 50% better retention than passive reading
- Forces your brain to work harder
- Creates stronger memory connections
- Identifies gaps in knowledge

**FocusUp Connection**:
- Use Quiz section for active recall
- Practice problems in different orders
- Test yourself daily

### 2️⃣ SPACED REPETITION
**What It Is**: Reviewing material at optimal intervals

**The Schedule**:
- Day 1: Learn new material
- Day 1 (later): First review
- Day 3: Second review
- Week 1: Third review
- Week 2: Fourth review
- Month 1: Fifth review

**Why It Works**:
- Fights the "forgetting curve"
- Long-term retention instead of cramming
- Permanent knowledge
- Less study time needed

**Tools for This**:
- **Anki**: Digital flashcard app
- **Quizlet**: Popular study app
- **FocusUp Quiz**: Revisit same topics regularly

### 3️⃣ MIND MAPPING
**What It Is**: Connecting concepts visually

**How to Create One**:
1. Write central topic in circle
2. Draw branches for main ideas
3. Add sub-branches for details
4. Use colors for different categories
5. Add images/drawings

**Why It Works**:
- Brain remembers images better than text
- Shows connections between concepts
- Helps organize information
- Great for essay writing

**Example**: 
```
              Photosynthesis
                   |
        ___________|___________
       |           |           |
    Inputs      Process     Outputs
       |           |           |
    H₂O,CO₂     Light Rx.   Glucose,O₂
```

### 4️⃣ INTERLEAVING
**What It Is**: Mixing different topics during study

**Old Way (Blocking)**:
- Study all quadratic equations
- Then all linear equations
- Then all polynomials

**New Way (Interleaving)**:
- Mix all equation types together
- Random order
- Forces deeper thinking

**Why It Works**:
- Better problem-solving skills
- Improves transfer to new situations
- More challenging = stronger learning
- Prevents false confidence

**How to Do It**:
- Randomize practice problems
- Mix topics in study sessions
- Quiz with mixed subjects
- Alternate between similar concepts

## 🆘 TROUBLESHOOTING

### Stats Not Updating?
- Refresh page: F5 or Cmd+R
- Check localStorage is enabled
- Try different browser

### Dark Mode Not Saving?
- Cookies enabled? (Settings)
- Try clearing cache first
- Re-select dark mode

### Timer Not Working?
- Close browser and reopen
- Try different browser
- Check volume is on

### Goals Not Saving?
- Check localStorage space
- Clear old data from other sites
- Try private/incognito mode
